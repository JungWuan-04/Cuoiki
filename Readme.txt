1. Giới thiệu
Tên dự án: Ứng dụng Gmail Flutter (Mô phỏng)  
Mục đích: Ứng dụng Flutter được phát triển cho bài tập cuối kỳ, mô phỏng hệ thống email đơn giản với các chức năng như đăng nhập, quản lý người dùng và hiển thị email.

2. Hướng dẫn cài đặt và chạy dự án
Yêu cầu:
- Flutter SDK mới nhất: https://flutter.dev/docs/get-started/install
- Android Studio hoặc Visual Studio Code (có plugin Flutter & Dart)
- Thiết bị thật hoặc trình giả lập Android/iOS

Cách thực hiện:
1. Giải nén thư mục dự án.
2. Mở terminal và điều hướng đến thư mục gốc:
   cd ten-thu-muc-du-an
3. Cài đặt các thư viện:
   flutter pub get
4. Chạy ứng dụng:
   flutter run

4. Thông tin máy chủ

Dữ liệu được xử lý cục bộ hoặc mô phỏng trong mã nguồn.

5. Các tính năng tùy chọn đã triển khai 
- Giao diện mô phỏng đơn giản giống Gmail.
- Lưu thông tin đăng nhập bằng shared_preferences.
- Hiển thị danh sách email mẫu, kèm người gửi và thời gian.
- Cấu trúc mã nguồn rõ ràng, chia theo widget riêng biệt.

6. Ghi chú thêm
- Nếu gặp lỗi khi build, vui lòng kiểm tra Flutter bằng:
  flutter doctor

- Nếu vẫn gặp lỗi, thử xóa thư mục `.dart_tool` và chạy:
  flutter pub get
